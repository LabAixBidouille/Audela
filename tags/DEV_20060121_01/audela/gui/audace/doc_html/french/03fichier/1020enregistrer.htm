Enregistrer

L'option Enregistrer se trouve dans le menu déroulant Fichier :

Par exemple, si après avoir chargé une image à l'écran, vous modifiez ses seuils de visualisation, vous lui appliquez un traitement, etc., en cliquant sur Enregistrer vous la sauvegarderez avec les changements sous le même nom.

Notez qu'Aud'ACE ne vous demande pas de lui communiquer le nom d'une nouvelle image, puisque l'image avec les changements apportés sera automatiquement enregistrée sous le même nom.
