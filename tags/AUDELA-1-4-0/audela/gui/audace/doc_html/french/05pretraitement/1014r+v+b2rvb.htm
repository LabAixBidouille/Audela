Conversion R+V+B ---> RVB

Le but de cette fonction est de reconstituer une image couleur (RVB) au format FITS (3 axes) à partir de 3 images acquises au travers de filtres dans les bandes de couleur rouge (R), verte (V) et bleue (B), également au format FITS.

Les noms des 3 images d'entrée doivent être par exemple :

  • nom_générique_entrée_1.fit
  • nom_générique_entrée_2.fit
  • nom_générique_entrée_3.fit

Il n'y a pas de règle particulière à suivre pour définir le nom de l'image de sortie.
