Configuration de la liaison

Pour régler les paramètres de connexion d'une liaison, il faut ouvrir la fenêtre de configuration des liaisons. Pour cela, vous devez aller dans le menu déroulant Configuration, puis cliquer sur Liaison... :

La fenêtre de configuration des liaisons se présente sous la forme suivante :

Vous pouvez constater que le logiciel supporte un certain nombre de liaisons. Il vous suffit de cliquer sur les différents onglets disponibles (AudiNet, EthernAude, GPhoto2, Port parallèle, PhotoPC, QuickAudine, QuickRemote, Port série) pour explorer les possibilités d'interfaçage.

Dans la partie inférieure de la fenêtre ci-dessus, vous constatez la présence, derrière une case à cocher, du message suivant : Connexion automatique de cette liaison au prochain démarrage

Si après avoir coché cette case, vous appuyez sur le bouton OK ou Appliquer, à la fermeture du logiciel, celui-ci vous demandera d'enregistrer la nouvelle configuration.

Au prochain démarrage du logiciel, vous n'aurez pas à reconfigurer votre liaison. Avec cette option, la liaison EthernAude (dans notre exemple) se connecte automatiquement en utilisant le paramétrage précédemment choisi.

Enfin cette fenêtre comporte 4 boutons dans sa partie inférieure dont les fonctions sont les suivantes :

  • OK : Applique le paramétrage choisi pour la liaison définie par l'onglet sélectionné et ferme la fenêtre de configuration des liaisons.
  • Appliquer : Applique le paramétrage choisi pour la liaison définie par l'onglet sélectionné et laisse la fenêtre de configuration des liaisons ouverte.
  • Aide : Permet d'accéder directement à l'aide pour la configuration de la liaison choisie par l'onglet sélectionné.
  • Fermer : Ferme la fenêtre de configuration des liaisons sans prendre en compte les paramètres modifiés.

Vous pouvez accéder au paramétrage de chaque liaison en cliquant ici.
