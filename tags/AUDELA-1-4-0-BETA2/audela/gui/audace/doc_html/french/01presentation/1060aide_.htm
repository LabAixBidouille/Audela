Aide

Comme tout logiciel digne de ce nom, Aud'ACE vous permet d'accéder à une aide complète par le menu déroulant Aide :

1. Sommaire

Un simple clic de souris sur Sommaire ouvre la fenêtre suivante pour naviguer dans l'arborescence de l'aide :

2. Index des fonctions

Par un simple clic sur Index des fonctions, vous avez accès à toutes les fonctions disponibles du logiciel. Elles sont généralement accompagnées d'une description plus ou moins détaillée.

3. Accès au Site Web

Vous avez deux possibilités pour accéder au Site Web :

Soit directement via une liaison Internet par l'option Site Web (Internet), mais attention il faut également avoir correctement configuré le Navigateur Web dans la boîte de dialogue des Editeurs (Configuration --> Editeurs) :

Soit en accédant à une copie locale du site web sur votre disque dur par l'option Site Web (copie sur le disque dur)... Cette copie doit impérativement être dans le répertoire \gui\audace\doc_html\web_site. Et bien sûr la rubrique Navigateur Web doit être correctement configurée dans la boîte de dialogue des Editeurs (Configuration --> Editeurs) :

4. Accès à la documentation sous forme de fichiers pdf

Certaines documentions sont uniquement accessibles sous la forme de fichiers pdf. Pas de problème, si vous avez correctement configuré Aud'ACE en définissant la localisation d'Acrobat Reader dans la rubrique Voir les notices (.pdf) (Configuration --> Editeurs), un simple clic de souris sur Notices (.pdf)... :

Et la boîte de dialogue suivante s'ouvre :

Dans notre exemple nous avons choisi de visualiser la notice décrivant la caméra MASCOT (Mini All Sky Cloud Observation Tool) de surveillance du ciel pour les VLT (Very Large Telescopes) à La Silla pilotée par Aud'ACE. Après avoir sélectionné le fichier mascot.pdf, il vous suffit d'appuyer sur le bouton Ouvrir. Acrobat Reader se lance et ouvre le document sélectionné. Une autre façon consiste à double cliquer sur l'icône du fichier choisi, là aussi Acrobat Reader se lance et ouvre automatiquement la notice désirée.

5. A propos de ...

Un simple clic de souris sur A propos de ... :

Et la fenêtre suivante s'ouvre :

Elle vous communique diverses informations comme la version du logiciel que vous êtes en train d'utiliser, la version de l'interpréteur Tcl/Tk trouvé, la date de mise à jour, la certitude que ce logiciel est libre et gratuit, le lien hypertexte qui permet une connexion directe au site officiel, le copyright, ainsi que les noms des différentes personnes ayant contribuées au développement et à la traduction, respectivement en anglais, en italien, en espagnol, en allemand et en danois de la présente distribution.

Vous disposez avec ce menu Aide de tous les éléments nécessaires pour utiliser correctement Aud'ACE. Il ne vous reste plus qu'à vous lancer.
