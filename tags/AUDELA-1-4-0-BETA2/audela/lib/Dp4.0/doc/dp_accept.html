dp_accept

Syntax 
 
dp_accept chanID
 
Comments
 
This command is used to let a server TCP channel listen for client connections.  Do not use this command on non-TCP channels. dp_accept blocks until a client is connected.
 
This command returns a list with the channel name of the new client/server connection and the IP address of the client which was accepted.
 
Examples
 
set cliChan [lindex [dp_accept $myServerChan] 0]
