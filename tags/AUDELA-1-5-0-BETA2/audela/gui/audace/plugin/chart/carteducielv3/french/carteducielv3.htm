Carte du Ciel Version 3

AudeLA peut communiquer avec "Carte du ciel" pour :

- Visualiser une carte correspondant à une image affichée dans Audela.

- Pointer un objet désigné dans "Carte du ciel" et déplacer le télescope vers cet objet

Au préalable, il faut configurer la communication entre AudeLA et Carte du ciel .

 

Configuration de "Carte du ciel"

  • Ouvrir le menu Setup->Configure The program
  • Sélectionner le menu System->Server
  • cocher "Use TCP/IP Server"
  • cocher "Client Connction Keep Alive
  • renseigner Server IP Interface : 127.0.0.1
  • renseigner Server IP Port : 3292
  • enregistrer les paramètres en cliquant sur le bouton "OK"

Configuration de "AudeLA"

  • Ouvrir le menu Configuration->Carte
  • selectionner l'onglet "Carte du ciel V3"
  • renseigner Serveur TCP/IP : 127.0.0.1
  • renseigner Port : 3292

Paramètres optionnels :

  • Cocher "champ fixe" pour imposer un champ de taille fixe dans Carte du Ciel. Préciser la taille du champ en renseignant la valeur en degrés, minutes, secondes. Si cette option n'est pas cochée, la taille du champ n'est pas modifiée.
    Exemple : 01d00m00s permet d'obtenir une carte avec un champ de 1 degré.
  • Renseigner "Recherche automatique" avec le répertoire du programme "Carte du ciel" pour permettre à AudeLA de lancer "Carte du ciel" s'il n'est pas déjà lancé.
  • Renseigner "Fichier à rechercher" et "A partir de" si l'on ne connait pas le répertoire du programme "Carte du ciel" et cliquer sur le bouton "Recherche automatique"

Visualiser une carte correspondant à une image affichée dans Audela

  • Ouvrir une image avec AudeLA avec le menu Fichier->Ouvrir
  • sélectionner le menu Analyse->Carte et la carte correspondant à l'image est affichée dans "Carte du ciel"


 

Pointer un objet désigné dans "Carte du ciel"

  • Sélectionner une monture dans AudeLA en cochant l'option "Raquette à l'écran" et en choisissant la raquette "superpad"
  • Sélectionner une étoile dans Carte du ciel

  • Cliquer sur le bouton "Carte" de la raquette pour récupérer l'étoile sélectionnée dans AudeLA

    le nom de l'étoile et ses coordonnées apparaissent dans la raquette.

  • Cliquer sur le bouton "GOTO" de la raquette pour déplacer le télescope vers l'étoile
