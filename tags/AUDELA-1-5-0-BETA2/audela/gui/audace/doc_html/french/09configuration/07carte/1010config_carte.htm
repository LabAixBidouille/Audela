Configuration de cartes

Pour régler les paramètres de connexion d'une carte, il faut ouvrir la fenêtre de configuration des cartes. Pour cela, vous devez aller dans le menu déroulant Configuration, puis cliquer sur Carte... :

La fenêtre de configuration des cartes se présente sous la forme suivante :

Vous pouvez constater que le logiciel supporte un certain nombre de cartes. Il vous suffit de cliquer sur les différents onglets disponibles (Carte du Ciel V2, Carte du Ciel V3, Guide) pour explorer les possibilités de pilotage.

Dans la partie inférieure de la fenêtre ci-dessus, vous constatez la présence, devant une case à cocher, du message suivant : Connexion automatique de cette carte au prochain démarrage

Si après avoir coché cette case, vous appuyez sur le bouton OK ou Appliquer, à la fermeture du logiciel, celui-ci vous demandera d'enregistrer la nouvelle configuration.

Au prochain démarrage du logiciel, vous n'aurez pas à reconfigurer votre carte. Avec cette option, la carte Carte du Ciel V2 (dans notre exemple) se connecte automatiquement en utilisant le paramétrage précédemment choisi.

Cette option est intéressante pour la plupart des utilisateurs car ils utilisent souvent la même carte. Elle leur économise quelques manipulations dans les menus.

Enfin cette fenêtre comporte 4 boutons dans sa partie inférieure dont les fonctions sont les suivantes :

  • OK : Applique le paramétrage choisi pour la carte définie par l'onglet sélectionné et ferme la fenêtre de configuration des cartes.
  • Appliquer : Applique le paramétrage choisi pour la carte définie par l'onglet sélectionné et laisse la fenêtre de configuration des cartes ouverte.
  • Aide : Permet d'accéder directement à l'aide pour la configuration de la carte choisie par l'onglet sélectionné.
  • Fermer : Ferme la fenêtre de configuration des cartes sans prendre en compte les paramètres modifiés.

Vous pouvez accéder au paramétrage de chaque carte en cliquant ici.
