Statistiques dans la fenêtre

Cette commande, lancée après le tracé d'un rectangle de sélection sur l'image avec la souris, affiche une fenêtre annexe fournissant les statistiques d'intensité :

  • le numéro de la visu,
  • le maximum,
  • le minimum,
  • la moyenne,
  • l'écart-type,
  • la moyenne du fond du ciel,
  • l'écart-type du fond du ciel.
