Fonctions de transfert

1. Linéaire

Ne privilégie aucune zone de niveaux de gris.

2. Logarithme

Accentue le contraste des zones sombres.

3. Exponentielle

Accentue le contraste des zones claires.

4. Arctangente

Accentue le contraste des zones moyennes.

Le paramètre doit être un nombre positif : Plus il est élevé et plus l'effet est accentué.
