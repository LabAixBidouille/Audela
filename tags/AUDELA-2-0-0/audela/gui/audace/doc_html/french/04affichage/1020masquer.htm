Masquer le panneau

Cette option du menu déroulant Affichage permet par un simple clic de la souris de faire disparaître l'outil en ligne (affiché sur la gauche de l'interface) et donc d'attribuer le maximum d'espace dans l'interface Aud'ACE pour l'affichage des images.

Le raccourci clavier F12 a été attribué à cette fonctionnalité.
