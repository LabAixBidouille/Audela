Fonctions de la librairie libgs

Liste des fonctions utilisables :

gs_version

Retourne le numéro de la version de la librairie libgs utilisée.

gs_guide version

Renvoie GUIDE6, GUIDE7 ou GUIDE8 en fonction de la version de GUIDE qui est utilisée.

gs_guide refresh

Rafraîchit l'heure courante de GUIDE.

gs_guide show

Permet d'afficher la fenêtre de GUIDE.

gs_guide hide

Permet de masquer la fenêtre de GUIDE.

gs_guide restore

Restaure le dernier état de la fenêtre de GUIDE.

gs_guide objet objet ?date?

Permet de pointer GUIDE sur un objet en utilisant l'heure système comme heure de référence. Cet objet peut être déclaré de plusieurs façons différentes :

  • Objet du catalogue Messier : M104
  • Objet du catalogue NGC : NGC891
  • Objet du catalogue IC : IC434
  • Cas des astéroïdes : A2 désigne l'astéroïde numéro 2.

Si l'on souhaite réaliser l'affichage de GUIDE pour une heure différente de celle du système, il suffit de le lui faire savoir, dans ce cas la commande est :

gs_guide objet M104 JD=2452447.30275

gs_guide coord ascension_droite déclinaison époque ?date?

Permet de pointer GUIDE sur une position. Il est impératif de fournir l'ascension droite, la déclinaison et l'époque.

Si l'on souhaite réaliser l'affichage de GUIDE pour une heure différente de celle du système, il suffit de le lui faire savoir, dans ce cas la commande est :

gs_guide coord {22h17m45.2s} {-3d16m23s} {J2000} JD=2452447.30275

gs_guide zoom zoom

Permet de fixer le niveau du zoom de la fenêtre de GUIDE. Ce paramètre varie de 1 à 20.
