Configuration de la raquette

Pour régler les paramètres de connexion d'une raquette, il faut ouvrir la fenêtre de configuration des raquettes. Pour cela, vous devez aller dans le menu déroulant Configuration, puis cliquer sur Raquette... :

La fenêtre de configuration des raquettes se présente sous la forme suivante :

Vous pouvez constater que le logiciel supporte un certain nombre de raquettes. Il vous suffit de cliquer sur les différents onglets disponibles (Raquette LX200, SuperPad, TelPad) pour explorer les possibilités de pilotage.

Cette fenêtre comporte 4 boutons dans sa partie inférieure dont les fonctions sont les suivantes :

  • OK : Applique le paramétrage choisi pour la raquette définie par l'onglet sélectionné et ferme la fenêtre de configuration des raquettes.
  • Appliquer : Applique le paramétrage choisi pour la raquette définie par l'onglet sélectionné et laisse la fenêtre de configuration des raquettes ouverte.
  • Aide : Permet d'accéder directement à l'aide pour la configuration de la raquette choisie par l'onglet sélectionné.
  • Fermer : Ferme la fenêtre de configuration des raquettes sans prendre en compte les paramètres modifiés.

Vous pouvez accéder au paramétrage de chaque raquette en cliquant ici.
