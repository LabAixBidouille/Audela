Compiler LibMC


1. Structure du dossier libmc

Après avoir téléchargé le fichier libmc.zip, décompresser le dans un dossier nommé libmc. On y trouvera alors les dossiers suivants :
  • html : contient la présente documentation
  • sources : contient les fichiers source (.c et .h) de libmc
  • os_linux_so : contient le batch de compilation pour gcc sous Linux.
  • os_win_vcpp_dll : contient le batch de compilation pour Microsoft C++ sous Windows.
  • hlin : contient les fichiers d'entête C de tcl pour compilation Linux.
  • hwin : contient les fichiers d'entête C de tcl pour compilation Windows.

2. Compiler libmc sous Linux avec gcc

On commencera par se placer dans le répertoire os_linux_so. Le batch de compilation se nomme makelib.gcc. Il faut d'abord rendre ce batch exécutable sous Linux. Pour cela, exécuter chmod +x makelib.gcc. Ensuite, suivre les instructions suivantes, dans l'ordre :
  • copier tous les fichiers présents dans le dossier source vers le dossier os_linux_so.
  • exécuter le batch makelib.gcc.
  • copier le fichier libmc.so dans le dossier binlin du logiciel AudeLA.

3. Compiler libmc sous Windows avec Microsoft C++

On commencera par se placer dans le répertoire os_win_vcpp_dll. Le projet senomme libmc.mdp. Suivre les instructions suivantes, dans l'ordre :
  • lancer le projet libmc.mdp.
  • sélectionner l'option de compilation Release (l'option Debug génère un plus gros fichier).
  • compiler en choisissant "construire libmc.dll".
  • le fichier libmc.dll est généré dans le dossier Release.
  • copier le fichier libmc.dll dans le dossier bin du logiciel AudeLA.
