Conversion de fichiers images au format FITS

Cette page décrit comment convertir au format FITS divers autres formats d'image. Pour cela, Aud'ACE utilise le programme "BifsConv" de Raoul Behrend. La liste des formats d'image pris en charge (notamment bmp, pic, tif, raw, etc.) figure sur cette page. Ce programme est fréquemment mis à jour par Raoul Behrend, le fichier bifsconv.exe (commun pour Windows et pour Linux) peut être téléchargé à l'adresse : http://obswww.unige.ch/~behrend/page_bif.html, fichier dont la version à mettre à jour se trouve dans le sous-répertoire bin du répertoire d'installation d'Aude'ACE.

Liste des fonctions utilisables :

bifsconv fichier options

Lance bifsconv pour un fichier placé dans le répertoire images (le nom de fichier doit comporter l'extension). Il est possible (voire nécessaire) de préciser certaines options selon le format source (consulter à ce propos la page web de bifsconv).

Exemple : bifsconv mire.tif

bifsconv_full chemin_fichier options

Idem que bifsconv (voir ci-dessus) sauf qu'il faut préciser le nom complet du fichier (permet de convertir un fichier qui ne serait pas dans le répertoire images Aud'ACE par défaut).

Exemple : bifsconv_full "C:/Astronomie/AudeLA SourceForge/audela/images/test2/test3/centre kac1310.gif"

convert_fits nom_generique extension ?répertoire?

Permet de convertir au format FITS une série d'images dont le format figure parmi ceux pris en charge par bifsconv.

Exemple : convert_fits mire_ .tif

Exemple : convert_fits mire_ .tif "C:/Astronomie/AudeLA SourceForge/audela/images/test2/test3"

convert_fits_all extension ?répertoire?

Permet de convertir au format FITS toutes les images du répertoire courant (parmi les formats pris en charge par bifsconv).

Exemple : convert_fits_all .tif

Exemple : convert_fits_all .tif "C:/Astronomie/AudeLA SourceForge/audela/images/test2/test3"

convert_fits_subdir extension ?répertoire?

Idem "convert_fits_all" avec de plus recherche de fichiers dans les sous-répertoires.

Exemple : convert_fits_subdir .tif

Exemple : convert_fits_subdir .cpa "C:/Astronomie/AudeLA SourceForge/audela/images/test2/test3"

loadima_nofits fichier ?repertoire?

Chargement et affichage d'une image qui ne serait pas au format FITS (parmi ceux pris en charge par bifsconv). Le nom de fichier doit comporter l'extension. Par défaut l'image est supposée située dans le répertoire images par défaut.

Exemple : loadima_nofits NGC1.cpa

Exemple : loadima_nofits NGC104.cpa "C:/Astronomie/AudeLA SourceForge/audela/images/test2/test3"
