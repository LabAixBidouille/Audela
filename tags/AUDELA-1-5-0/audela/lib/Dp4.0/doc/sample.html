Sample Applications

As an trivial example of what Tcl-DP can do, we have included several samples of how one can use Tcl-DP. In the examples subdirectory, there is:

  • A Tk whiteboard application
  • An FTP-like protocol for file transfer between two Tcl interpreters
  • A group text conferencing system and, of course,
  • Tic-Tac-Toe

Note that the whiteboard and conferencing system would both benefit by switching the network protocol from TCP to IPM but since dp_MakeRPC{Server,Client} both use TCP, that task is left to the reader.
