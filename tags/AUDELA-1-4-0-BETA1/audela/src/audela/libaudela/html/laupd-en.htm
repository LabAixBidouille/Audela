update 20031128

Diary of updates

2003-11-28 :

  • Add the third order possibility of calculation for "buf1 xy2radec" and "buf1 radec2xy" if PV* keywords are presents in the FITS header.
  • Few orrections in  function CmdPortTalk in libstd.cpp suggested by R. Petitdemange and J. Michelet.
  • All strings are resized to 1000 characters in buf_tcl.cpp.

2003-11-11 :

  • Add the new function "buf1 save3d" that splits a 2D image into a 3D image. Naxis3 value is a parameter and the naxis2 is splited.
  • Add the new function "visu1 disp2buf" that creates a buffer from the image displayed.

2003-11-09 :

  • Add the new function "fitsheader" that returns a list of keywords and ther values in a header of a FITS file.
  • Add the new function "fitsconvert3d" that converts a series of FITS files 2D (naxis1, naxis2) into only one FITS file 3D (naxis1, naxis2, naxis3).
  • Correction of a bug of number of arguments in the function "buf1 load3d".

2003-10-30 :

  • Add the second order possibility of calculation for "buf1 xy2radec" and "buf1 radec2xy" if PV* keywords are presents in the FITS header.
  • Correction of a bug when no arguments lie after ttscript2 function.

2003-10-25 :

  • Add a NULL pointer test on pix and p_ast pointers in functions CBuffer::xy2radec and CBuffer::radec2xy in buffer_tcl.cpp.

2003-10-19 :

  • Add the new function "buf1porttalk" to use the included porttalk librarie for Win32. Create function CmdPortTalk in libstd.cpp. Modify the project file for Win32 compilation. (made by Michel Pujol)

2003-08-30 :

  • Add the new function "buf1 load3d" to load a plane or all the planes of an image defined with NAXIS=3. Create functions CBuffer::Load3d and cmdLoad3d in buffer_tcl.cpp.

2003-07-29 :

  • Add new values for "visu1 zoom". One can now use 0.125, 0.25, 0.5, 1, 2, 4 and 8. Corrections in the function CVisu::UpdateDisplay, CVisu::SetZoom, CVisu::GetZoom in  cvisu.cpp and cmdVisuZoom in visu_tcl.cpp.

2003-07-04 :

  • Corrections in the function CBuffer::Load and CBuffer::Save(file cbuffer.cpp). Freeing of pointer of libtt are properly done when an error occur.

2003-07-02 :

  • Minor correction in the function CmdHostaddress (file libstd.cpp).

2003-02-28 :

  • A new function "buf1 A_starlist" written by Bogumil and Dorota (Poland) is implanted.

2003-02-17 :

  • Many obsolete source files are destroyed, mainly for camera and telescope internal drivers. From now, all drivers of cam or tel must be externals. Following source files were destroyed : caudine.cpp, ccamera.cpp, chisis11.cpp, chisis22.cpp, chisis24.cpp, clx200.cpp, couranos.cpp, csercom.cpp, cst7.cpp, ctelesc.cpp, cultima.cpp, robert.cpp, angles.cpp, animate.cpp, ephem.cpp. Following source files were modified : pool_tcl.cpp, libstd.cpp, tel_tcl.cpp, cam_tcl.cpp.

  • Modification of keywords of astrometry in the file cbuffer.cpp.

2002-12-30 :

  • Add function ping (libstd.cpp).
  • Add function hostaddress (libstd.cpp).
