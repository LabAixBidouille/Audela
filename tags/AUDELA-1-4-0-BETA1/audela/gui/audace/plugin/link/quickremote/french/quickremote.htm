Liaison par QuickRemote

Description

Le boîtier QuickRemote permet de piloter un port de sortie constitués de 8 bits parallèles via une prise USB d'un micro-ordinateur. Les bits peuvent être utilisés pour piloter un ou plusieurs périphériques.

Exemple d'utilisation :

bit 0 : Commande la longue pose d'un APN.
bit 1 : Commande la longue pose d'une WebCam.
bits 2,3 : Commande un focuser (on/off, sens).
bits 4,5,6,7 : Commande la raquette d'un telescope à 4 fils.

Commandes génériques des objets link

::link::create quickremote usb_index

Résultat :

La commande crée le nouveau QuickRemote et retourne un numéro du QuickRemote non nul. Elle retourne une valeur nulle si la création a échoué.

Paramètre usb_index (défaut = 0)

usb_index est l'index du port USB attribué par le système d'exploitation au QuickRemote quand il est branché sur l'ordinateur. La commande "::link::listavailable quikremote" permet de connaître les index des QuickRemotes branchés sur l'ordinateur.

Exemple :

# je cree un quickremote
set linkno [::link::create quickremote 0]
if { $linkno != 0 } {
	# je met le bit 0 à la valeur 1
	quickremote$linkno bit 0 1
}
 

::link::delete linkno

Résultat :

La commande supprime le QuickRemote numéro "linkno"

Exemple :

# je cree un quickremote
set linkno [::link::create quickremote 0]
if { $linkno != 0 } {
	# je supprime le quickremote 
	::link::delete $linkno
}
 

Commandes spécifiques

quickremote1 char {0 ... 255}

Si aucune valeur n'est précisée, la commande retourne la valeur actuelle des bits du port de sortie.

Si une valeur entre 0 et 255 est précisée, la commande affecte la valeur au port de sortie.

Exemple : 

set a [quickremote1 char]

::console::disp $a
3                   # les bits 0 et 1 valent 1 , les autres bits valent 0

quickremote1 char 4
set a [quickremote1 char]

::console::disp $a
4                   # le bit 2 vaut 1, les autres bits valent 0

quickremote1 bit b {0|1}

Si aucune valeur n'est précisée, la commande retourne la valeur actuelle du bit numéro "b" , b étant une valeur entre 0 et 7.

Si une valeur 0 ou 1 est précisée, la commande affecte la valeur 0 ou 1 au bit numéro b.

set a [quickremote1 bit 0] 
::console::disp $a
0                            # le bit 0  vaut 0
quickremote1 bit 0 1         
set a [quickremote1 bit 0] 
::console::disp $a
0                            # le bit 0 vaut 1
