Fonctions de traitement d'images stellaires

Liste des fonctions utilisables :

compute_stellaire brutes ?noirs? ?PLUs? ?noirsdePLU? ?options?

Prétraitement complet d'une série d'images brutes ; les images prétraitées sont triées selon la finesse des étoiles, et l'on conserve un certain pourcentage (par défaut 80%) pour une addition sigma-kappa. L'image résultante est affichée dans le buffer Aud'ACE.

Options :

  • -buf : spécifie le buffer monochrome dans lequel on charge l'image (par défaut, c'est le buffer Aud'ACE).
  • -rep : spécifie le répertoire dans lequel travailler.
  • -ext : extension de l'image.
  • -polyNo : spécifie le plan couleurs à charger (par défaut le 1er, qui est le seul disponible pour des images monochromes).
  • -tri : pourcentage d'images à conserver après tri selon la finesse des étoiles.
