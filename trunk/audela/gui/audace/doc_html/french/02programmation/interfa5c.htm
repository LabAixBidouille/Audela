Liste de toutes les fonctions

Nous donnons la liste de toutes les fonctions disponibles dans AudeLA, plus quelques unes de Tcl et Tk qui sont très utiles. En ce qui concerne les fonctions associées à des objets de AudeLA, elles sont présentées en fonction de l'objet numéro 1. Les fonctions ont été rassemblées par thème et sont suivies de leur origine:

  • [TC] : Fonction Tcl, disponible dans toutes les applications AudeLA
  • [AL] : Fonction AudeLA, disponible dans toutes les applications AudeLA
  • [AC] : Fonction Aud'ACE, disponible uniquement avec l'interface Aud'ACE
  • [MC] : Fonction de la librairie d'extension MC, disponible dans toutes les applications AudeLA
  • [RG] : Fonction de la librairie d'extension RGB, disponible dans toutes les applications AudeLA
  • [GZ] : Fonction de la librairie d'extension GZIP, disponible dans toutes les applications AudeLA
  • [GS] : Fonction de la librairie d'extension G.S.L., disponible dans toutes les applications AudeLA
  • [DC] : Fonction d'une librairie de driver de caméra
  • [DT] : Fonction d'une librairie de driver de télescope
  • [DP] : Fonction de la librairie d'extension DP
  • [BLT] : Fonction de la librairie d'extension BLT
  • [BM] : Fonction de la librairie d'extension LibBM
  • [JM] : Fonction de la librairie d'extension LibJM
  • [GS] : Fonction de la librairie d'extension LibGS
