Configuration de la monture

Pour régler les paramètres de connexion d'une monture, il faut ouvrir la fenêtre de configuration des montures. Pour cela, vous devez aller dans le menu déroulant Configuration, puis cliquer sur Monture... :

La fenêtre de configuration des montures se présente sous la forme suivante :

Vous pouvez constater que le logiciel supporte un certain nombre de montures. Il vous suffit de cliquer sur les différents onglets disponibles (LX200, Ouranos, AudeCom, ComPad, AvrCom, TelCom, LXnet, Temma, MCMT) pour explorer les possibilités de pilotage.

Dans la partie inférieure de la fenêtre ci-dessus, vous constatez la présence, derrière une case à cocher, du message suivant : Connexion automatique de cette monture au prochain démarrage

Si après avoir coché cette case, vous appuyez sur le bouton OK ou Appliquer, à la fermeture du logiciel, celui-ci vous demandera d'enregistrer la nouvelle configuration.

Au prochain démarrage du logiciel, vous n'aurez pas à reconfigurer votre monture. Avec cette option, la monture LX200 (dans notre exemple) se connecte automatiquement en utilisant le paramétrage précédemment choisi.

Vous pouvez vérifier ceci en notant après l'ouverture du logiciel la présence de lx200 (pour notre exemple) en face de Monture dans la partie inférieure droite de la fenêtre principale (voir l'exemple ci-dessous).

Cette option est intéressante pour la plupart des utilisateurs car ils ne possèdent généralement qu'une seule monture. Elle leur économise quelques manipulations dans les menus.

Enfin cette fenêtre comporte 4 boutons dans sa partie inférieure dont les fonctions sont les suivantes :

  • OK : Applique le paramétrage choisi pour la monture définie par l'onglet sélectionné et ferme la fenêtre de configuration des montures.
  • Appliquer : Applique le paramétrage choisi pour la monture définie par l'onglet sélectionné et laisse la fenêtre de configuration des montures ouverte.
  • Aide : Permet d'accéder directement à l'aide pour la configuration de la monture choisie par l'onglet sélectionné.
  • Fermer : Ferme la fenêtre de configuration des montures sans prendre en compte les paramètres modifiés.

Vous pouvez accéder au paramétrage de chaque monture en cliquant ici.
