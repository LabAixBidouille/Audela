Configuration de l'équipement

Pour régler les paramètres de connexion d'un équipement, il faut ouvrir la fenêtre de configuration des équipements. Pour cela, vous devez aller dans le menu déroulant Configuration, puis cliquer sur Equipement... :

La fenêtre de configuration des équipements se présente sous la forme suivante :

Vous pouvez constater que le logiciel supporte un certain nombre d'équipements. Il vous suffit de cliquer sur les différents onglets disponibles (Roue à filtres) pour explorer les possibilités de pilotage.

Dans la partie inférieure de la fenêtre ci-dessus, vous constatez la présence, derrière une case à cocher, du message suivant : Connexion automatique de cet équipement au prochain démarrage

Si après avoir coché cette case, vous appuyez sur le bouton OK ou Appliquer, à la fermeture du logiciel, celui-ci vous demandera d'enregistrer la nouvelle configuration.

Au prochain démarrage du logiciel, vous n'aurez pas à reconfigurer votre équipement. Avec cette option, l'équipement Roue à filtres (dans notre exemple) se connecte automatiquement en utilisant le paramétrage précédemment choisi.

Cette option est intéressante pour la plupart des utilisateurs car ils utilisent souvent le même équipement. Elle leur économise quelques manipulations dans les menus.

Enfin cette fenêtre comporte 4 boutons dans sa partie inférieure dont les fonctions sont les suivantes :

  • OK : Applique le paramétrage choisi pour l'équipement défini par l'onglet sélectionné et ferme la fenêtre de configuration des équipements.
  • Appliquer : Applique le paramétrage choisi pour l'équipement défini par l'onglet sélectionné et laisse la fenêtre de configuration des équipements ouverte.
  • Aide : Permet d'accéder directement à l'aide pour la configuration de l'équipement choisi par l'onglet sélectionné.
  • Fermer : Ferme la fenêtre de configuration des équipements sans prendre en compte les paramètres modifiés.

Vous pouvez accéder au paramétrage de chaque équipement en cliquant ici.
