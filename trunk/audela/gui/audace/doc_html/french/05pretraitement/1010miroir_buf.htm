Miroir

A la différence avec les commandes Affichage/Miroir horizontal et Affichage/Miroir vertical qui n'agissent que sur l'affichage de l'image, les commandes suivante modifient le buffer de l'image.

1. Miroir vertical

Inversion gauche/droite dans le buffer.

2. Miroir horizontal

Inversion haut/bas dans le buffer.

3. Miroir diagonal

Inversion des dimensions de l'image dans le buffer : Equivaut à une rotation de 90° dans le sens inverse des aiguilles d'une montre.
