Site Web

Vous avez deux possibilités pour accéder au Site Web :

1. Site Web (Internet)

Soit directement via une liaison Internet par l'option Site Web (Internet), mais attention il faut également avoir correctement configuré le Navigateur Web dans la boîte de dialogue des Editeurs (Configuration --> Editeurs) :

2. Site Web (copie sur le disque dur)

Soit en accédant à une copie locale du site web sur votre disque dur par l'option Site Web (copie sur le disque dur)... Cette copie doit impérativement être dans le répertoire \gui\audace\doc_html\web_site. Et bien sûr la rubrique Navigateur Web doit être correctement configurée dans la boîte de dialogue des Editeurs (Configuration --> Editeurs) :
