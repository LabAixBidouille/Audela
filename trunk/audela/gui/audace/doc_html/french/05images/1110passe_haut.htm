Filtre passe-haut

Lien vers le manuel utilisateur de la librairie libtt pour le prétraitement d'images astronomiques.

Ce filtrage passe-haut est de type linéaire, et consiste à attribuer à chaque pixel de l'image résultat la somme pondérée de quelques pixels du voisinage. Les coefficients sont regroupés en un noyau dont la taille définit celle du voisinage. Cette opération s'appelle une convolution.
Les coefficients de ce filtre permettent d'atténuer les basses fréquences spatiales de l'image (détails grossiers se traduisant par une variation lente de luminosité de l'image), et d'en conserver les hautes fréquences. L'atténuation des basses fréquences croit avec la taille du noyau.
Les coefficients des noyaux suivants sont fournis pour une valeur du coefficient d'efficacité valant 0 ( efficacité maximale )
