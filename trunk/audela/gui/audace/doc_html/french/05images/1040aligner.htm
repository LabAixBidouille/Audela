2. Aligner automatiquement

2.1 Principe

Cette fonction exécute automatiquement les étapes suivantes :

  • Calcul des translations à effectuer pour que le contenu des images se recouvrent au mieux avec une précision inférieure au pixel.
  • Translation effective de chaque image.
  • Recadrage des images pour éliminer les zones sans recouvrement.

2.2 Quelles images faut-il aligner ?

La réponse dépend de l'origine des images :

  • Les images issues de caméras CCD peuvent être alignées à l'issue du prétraitement, après correction avec les images de Précharge, de Noir et de P.L.U.
  • Les images issues des APN doivent être converties en images couleurs RVB avant d'être alignées pour éviter les inconvénients induits par le codage sous forme de matrice de Bayer.

2.3 Paramétrage

Il suffit de cocher les images à aligner sur l'image de référence (à saisir dans la ligne image_ref) et d'indiquer le nom générique des images de sortie.

Les images recadrées portent le nom générique suivi d'un index débutant à 1.

Après vérification de la qualité de l'alignement, il suffit de les additionner, de faire la moyenne, etc. avec l'une des commandes Images/Combiner.
