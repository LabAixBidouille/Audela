Focaliseur USB_FOCUS

Pour de plus amples informations sur ce focaliseur voir le site de USB_Focus.

1. Connexion

Si port série n'est pas connecté (avant le lancement ou après un arrêt), toutes les commandes sont inhibées à l'exception du choix du port :

Après lancement ou lors de l'ouverture du panneau (si l'équipement n'a pas été arrêté), toutes les commandes sont libérées :

Les paramètres sont renseignés avec les valeurs lues dans le microprocesseur du focuser.

2. Informations et commandes disponibles

Les quatre groupes de commandes sont les suivants :

  • Logiciel du microprocesseur : le N° de version n'apparaît qu'après connexion. Un bouton RESET permet de rétablir les paramètres "usine".
  • Réglages du moteur : ils permettent un réglage fin adapté au montage de l'utilisateur. Sont disponibles :
    • La course maximale, exprimée en pas. Par défaut ce nombre est de 65535. Un bouton SET permet de modifier ce réglage.
    • La vitese de déplacement : un sélecteur permet de choisir une valeur entre 2 (la plus lente) et 9 (la plus rapide).
    • L'incrément : à chosir entre demi-pas ou pas entier.
    • Le sens de rotation : à choisir entre le sens des aiguilles d'une montre ou l'inverse.
  • Réglages de la compensation liée à la température : la température affichée est celle détectée par le capteur externe au boîtier. Un sélecteur permet d'activer ou non la compensation à effectuer lors de variations de température. Sont à indiquer le coefficient (pas /°C) et le sensibilité (seuil en dessous duquel il n'y a pas de correction). Le coefficient (pas le seuil) peut être positif ou négatif. Tant que le mode de compensation automatique est enclenché, les autres commandes sont inhibées.
  • Déplacement : La position actuelle est affichée. Deux modes de déplacement sont utilisables :
    • mode absolue : indiquer la position à atteindre puis appuyer sur GOTO.
    • mode incrémental : indique le déplacement à effectuer, en nombre de pas, et sélectionner le sens {+|-}.
    Durant le déplacement, les commandes sont inhibées.

3. Contrôle des saisies

  • nombre maximum de pas : nombre entier positif inférieur à 65535,
  • coefficient de compensation : nombre entier positif ou négatif, inférieur à ???,
  • seuiil de sensibilité : nombre entier positif inférieur à ???,
  • position absolue : nombre entier positif inférieur au nombre maximum de pas,
  • pas de déplacement : nombre entier positif ; le déplacement réel ne peut être supérieur au nombre maximum,
