Base de données des Images du Pic

Intro

     Il faut se referer de toute facon a la documentation de Bddimages pour tout
     ce qui concerne son fonctionnement.
     Les informations qui suivent ont pour but de faciliter l'installation des 
     PC de traitement et d acquisition, il est preferable de suivre chaque 
     etape pour plus de simplicité et de rapidité.

Preparation des Bases SQL

     == PHPMYADMIN ==
     Firefox -> http://localhost/phpmyadmin/

     == Creer la base ==
     Cliquer sur Databases -> Create database
     Taper : bdi_pic

     == Gerer les utilisateurs ==
     Cliquer sur Users
     
     Cocher tous les utilisateurs sauf root @ localhost
     puis supprimer tout le monde dans la partie "Remove selected users"
     Go
     
     Add user
     user name : bdi
     Host : Any host
     password : XXX
     
     Dans "Global privileges"
     cocher "Reload"
     
     cliquer sur Add user

     == Gerer les droits de l utilisateur bdi ==
     Cliquer sur Users
     Puis Edit Privileges correspondant a 'bdi'
     Dans la partie "Database-specific privileges"
     
     Add privileges on the following database : 
     taper dans la case a cette effet : bdi_%
     
     Puis dans la page qui vient de s afficher : 
     "Database-specific privileges"
     
     cliquer sur "Check All"

     == Voir les privileges utilisateurs ==
     Cliquer sur Users
     
     la liste doit ressembler a ca : 
     
     User   Host       Password  Global privileges  Grant
     bdi    %          Yes       RELOAD             No   
     root   localhost  Yes       ALL PRIVILEGES     Yes

Preparation des Disques Durs

     == Yast2 ==
     si les partitions ne sont pas toutes bien identifiées et montable par l utilisateur,
     aller dans Yast2 -> Partitionnement -> mode expert et gerer comme il se doit 
     
     == La partition bddimages ==
     sur bronto, /dev/sdc1 doit pointer sur /bddimages
     cette partition de 2 To peut accueillir plusieurs archives de donnees.
     
     == La partition d'image Audela ==
     la partition /dev/sdb1 pointant sur /raw accueille les images dites "brutes"
     dans laquelle on peut voir chaque repertoire qui sont les nuits d'observation.
     Durant chaque nuit d observation le logiciel Audela va creer un nouveau repertoire
     du nom de la nuit qui va suivre.
     Pour se faire chaque jour apres midi, il faut redemarrer Audela.
