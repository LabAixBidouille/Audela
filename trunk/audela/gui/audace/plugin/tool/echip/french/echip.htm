Paramètres électroniques

1. Fonctions

Cet outil constitue l'interface de la fonction electronic_chip qui permet de définir les paramètres électroniques, notamment le gain, le bruit de lecture et le coefficient thermique d'une caméra.

2. Interface

L'interface comprend une ligne pour indiquer le nom générique des images de précharge (offset), de noir (dark) et de P.L.U. (flat) nécessaires pour effectuer les calculs.
Le plus simple est de désigner l'image avec le bouton '...' : Le nom générique et le nombre d'images sont affichés automatiquement après vérification qu'il s'agit bien d'une série.

Pour éviter un échec, il convient de veiller à ce que :

  • Les mots clés EXPTIME ou EXPOSURE figurent dans les en-têtes des images, faute de quoi le temps d'exposition sera pris égal à 1 seconde.
  • Toutes les images concernées soient dans le même répertoire.
  • Ce répertoire doit être désigné dans le menu Configuration/Répertoires.
  • Configuration/Fichiers image doit désigner l'extension des images.

Rappel : Il faut au moins deux images de précharge, deux images de P.L.U. une quinzaine de noir de 0.5 seconde jusqu'à 10 secondes.

Pour définir le temps de pose maximum, il faut renseigner la Capacité à la saturation.

Si la caméra dispose d'un obturateur mécanique et si l'on veut visualiser son effet, il faut cocher la case correspondante. Dans ce cas il est préférable d'avoir une quinzaine d'images de P.L.U.
Les résultats sont affichés automatiquement et le détail des calculs est indiqué dans la Console.

Le Temps de pose critique est celui en-dessous duquel le bruit de lecture prédomine et au-dessus duquel le bruit thermique l'emporte sur le bruit de lecture : Il y aurait avantage à refroidir la caméra.

Le Temps de pose maximum correspond au temps nécessaire pour saturer le dark, il n'y a donc plus de place pour de l'information.
