Mise à jour AudeLA

Cet outil de télécharger des mises à jour de AudeLA qui sont disponibles sur Internet.

2) Voir le contenu d'un fichier kit

3.1) Fabrication de mise à jour d'un plugin

3.2) Fabrication de mise à jour du coeur d'AudeLA

1) Procédure de mise à jour de AudeLA

1) D'abord s'assurer que l'ordinateur est connecté à Internet .

2) Lancer AudeLA et selectionner l'outil de Mise à jour d'AudeLA (menu Outils->Mise à jour d'AudeLA)

Sélectionner l'onglet "Télécharger" et cliquer sur le bouton "GO Audela" pour voir les mises à jours disponibles sur le site Internet de AudeLA.

3) cliquer sur le nom d'un fichier (exemple: acqapn-1.0.kit) pour démarrer le téléchargement et l'installation automatique.

Un compte-rendu est affiché à la fin de l'installation :

4) redémarrer AudeLA pour prendre en compte l'installation.

2) Voir le contenu d'un fichier kit

Il est possible de consulter le contenu d'un fichier kit depuis l'onglet "Mises à jour téléchargées": sélectionner un kit puis cliquer sur le bouton "Voir". Le contenu du fichier kit est affiché dans la console.

 

 

3) Fabrication des mises à jour (pour les développeurs)

Il y a 3 sortes de mise à jour de AudeLA :

  • la mise à jour de module complémentaire (plugin)

    Cette mise à jour concerne les plugins qui sont dans le répertoire audela/gui/audace/plugin.

  • la mise à jour du coeur de AudeLA

    Cette mise à jour concerne tous les fichiers de AudeLa qui sont dans les répertoires audela/bin et audela/gui (excepté le répertoire des plugins).

    c'est une mise à jour incrémentale.

  • la mise à jour des librairies TCL

    Cette mise à jour concerne les librairies TCL qui sont dans le répertoire audela/lib

 

2.1) Fabrication de mise à jour d'un plugin

1) Lors des modifications du plugin, veiller à incrémenter le numéro de version du plugin pour que cette nouvelle version soit différentiée des précédentes.

  • modifier le numéro de version dans le fichier pkgindex.tcl

Exemple : 

package ifneeded acqapn 1.1 ...
  • et modifier le numéro de version dans le fichier principal du plugin . Eventuellement, on peut ajouter le numero de version de AudeLA requis pour ce plugin.

Exemple : 

namespace eval ::acqapn {
   package provide acqapn 1.1
   package require audela 1.4.0

2) Lancer AudeLA et selectionner l'outil de Mise à jour d'AudeLA (menu Outils->Mise à jour d'AudeLA)

Sélectionner l'onglet "Mise à jour installées" et sélectionner le plugin

3) Cliquer sur le bouton "Faire un kit" . Un compte-rendu est affiché à la fin de la fabrication :

4) Sélectionner l'onglet "Mise à jour téléchargée" pour constater que le nouveau fichier est présent.

 

5) Transférer le fichier sur le site Internet d'AudeLA (ou un site personnel) par FTP.

Exemple :

ftp ftp.audela.org
login :    michel
password : xxxxxx

cd /plugins
binary

send acqapn-1.1.kit

6) Sélectionner l'onglet "Télécharger" et cliquer sur le bouton "GO Audela" pour constater que le nouveau fichier acqapn-1.1.kit est publiée sur le site Internet de AudeLA
