Robotic Observatory Software

Ce plugin a pour but de communiquer avec un système de télescope robotique. Ce système peut être distant et public, comme il peut être local et privé. L'écriture de ce plugin est grossière, inachevée, son existence dans le projet audela-dev nous sert a le développer. Si vous n'êtes pas développeur, s'il vous plait désactivez ce plugin dans la fenêtre "Sélection des outils".

Prérequis

Bddimages prêt à fonctionner.
