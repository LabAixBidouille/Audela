Cartes du Ciel Version 3

Aud'ACE peut communiquer avec "Cartes du Ciel" pour :

- Visualiser une carte correspondant à une image affichée dans Aud'ACE.

- Pointer un objet désigné dans "Cartes du Ciel" et déplacer le télescope vers cet objet.

Au préalable, il faut configurer la communication entre Aud'ACE et Cartes du Ciel.
 

Configuration de "Cartes du Ciel"

  • Ouvrir le menu Configuration --> Système.
  • Sélectionner l'onglet Serveur.
  • Cocher Utiliser le serveur.
  • Cocher Garde la connexion client.
  • Renseigner Interface réseau : 127.0.0.1
  • Renseigner Port : 3292
  • Enregistrer les paramètres en cliquant sur le bouton OK.

Configuration de "Aud'ACE"

  • Ouvrir le menu Configuration->Carte.
  • Sélectionner l'onglet Carte du Ciel V3.
  • Renseigner Serveur TCP/IP : 127.0.0.1
  • Renseigner Port : 3292
  • Enregistrer les paramètres en cliquant sur le bouton OK.

Paramètres optionnels :

  • Cocher Champ fixe pour imposer un champ de taille fixe dans Cartes du Ciel. Préciser la taille du champ en renseignant la valeur en degrés, minutes et secondes d'angle. Si cette option n'est pas cochée, la taille du champ n'est pas modifiée.
    Exemple : 01d00m00s permet d'obtenir une carte avec un champ de 1 degré.
  • Renseigner Recherche (Cartes du ciel V3) avec le chemin complet du programme Cartes du Ciel pour permettre à Aud'ACE de lancer Cartes du Ciel, s'il n'est pas déjà lancé.

Visualiser une carte correspondant à une image affichée dans Aud'ACE

  • Ouvrir une image dans Aud'ACE avec le menu Fichier->Ouvrir.
  • Sélectionner le menu Analyse->Carte, la carte correspondant à l'image est affichée dans Cartes du Ciel".


 

Pointer un objet désigné dans "Cartes du Ciel"

  • Sélectionner une monture dans Aud'ACE avec le menu Configuration --> Monture en cochant l'option Raquette à l'écran et en choisissant la raquette superpad. Ensuite cliquer sur le bouton OK.
  • Sélectionner avec la souris l'étoile à pointer avec le télescope dans Cartes du Ciel.

  • Cliquer sur le bouton Carte de la raquette SuperPad pour récupérer dans Aud'ACE les coordonnées de l'étoile sélectionnée dans Cartes du Ciel.

  •  Le nom de l'étoile (Alpha Vir) et ses coordonnées (Ascension Droite et Déclinaison) apparaissent dans la raquette.

  • Cliquer sur le bouton GOTO de la raquette pour déplacer le télescope vers l'étoile.
