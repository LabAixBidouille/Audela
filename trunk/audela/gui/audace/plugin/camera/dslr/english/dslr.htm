Digital Camera

Plugin for Canon USB Digital Camera.

User manual

Before a first use with AudeLA and WINDOWS, it is necessary to install "libusb-win32" driver :

First, downloads libusb-win32-filter-bin-0.1.10.1.exe from http://libusb-win32.sourceforge.net
Direct link : http://downloads.sourceforge.net/libusb-win32/libusb-win32-filter-bin-0.1.10.1.exe

Then executes this file with default options.

Finally, starts Audela and connects your 20D with the "Digital camera" configuration panel.

Attention : The screen control of digital camera must be off when AudeLA sends commands to the camera.
