Windows - controlling parallel port.

  • Win95/98 or AllowIo on WinNT systems

Use libcam_out function (from ...common\libcam\util.c - standard implementation of controlling parallel port in AudeLa). Before compiling libwebcam define OS_WIN_USE_LPT_OLD_STYLE (in camera.h).

#define OS_WIN_USE_LPT_OLD_STYLE

 

  • WinXP and others WinNT systems (without AllowIo)

Don't define OS_WIN_USE_LPT_OLD_STYLE (in camera.h) -  parallel port will be used like printer port ("lpt1 "). In this case "null printer" modified plug is needed - for details please see Null Printer Adapter.

//#define OS_WIN_USE_LPT_OLD_STYLE

 

*find more about AllowIo with www.google.com.
