This page needs to be translated into English.

We are looking for translators with English or American English as mother tongue to help us translate the help files of Audela. Please contact me (christian.jasinski@gmail.com) if you can help us in your spare time.

If you have installed Audela on your hard disk, the French help page (av4l.htm) to be translated can be found in the "audace\plugin\tool\av4l\french\" folder. Once you have translated a page, please note that the name of the file must remain the same as the French name. Put the translated English file in the "audace\plugin\tool\av4l\english\" folder. Please send me the page you translated (see above e-mail address) so that it is included in the next Audela version.

Thank you!

In the meantime, you can have a look at the corresponding French help chapter.
