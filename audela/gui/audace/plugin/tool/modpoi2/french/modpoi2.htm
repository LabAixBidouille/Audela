Modèle de pointage V2

Le pointage d'un télescope est réalisé à partir de coordonnées lues dans des catalogues. Ces coordonnées ne sont pas exactement celles à envoyer au télescope. Il faut d'abord corriger les coordonnées catalogues en coordonnées apparentes qui prennent en compte les mouvements de la Terre. Il faut ensuite corriger les coordonnées apparentes en coordonnées télescopes pour prendre en compte les imperfections électromécaniques du télescope, ce qui exige un apprentissage. Pour cela, on effectue des pointages d'étoiles du catalogue Hipparcos, réputé pour sa grande précision. Le pointage pour l'apprentissage doit être fait régulièrement sur le ciel.

La sélection du menu Télescope/Modèle de pointage V2 ouvre la fenêtre principale de l'outil :

Si un modèle a déjà été créé, il peut être appelé, modifié et enregistré par le menu Fichier/Ouvrir. Dans le cas contraire il est conseillé de suivre les étapes décrites ci-dessous.

1. Processus de définition de l'horizon

Préalable impératif : Définir la position du télescope dans la fenêtre Configuration/Position de l'observateur sinon Aud'ACE considèrera que vous êtes à l'OHP !
Le menu Edition/Modifier l'horizon ouvre une nouvelle fenêtre permettant de définir un horizon spécifique.
Définir un nom de l'horizon comme monHorizon ou hautFolin. L'inscrire dans la fenêtre qui s'ouvre en appuyant sur le bouton Nouvel horizon et sélectionner le format des données {ALTAZ|HADEC}.
Valider en appuyant sur OK. Retour à la fenêtre principale pour définir les coordonnées de l'horizon choisi dans le format choisi.
Ici le modèle d'horizon est ALTAZ. Dans la fenêtre Coordonnées, entrer la liste des couples de valeurs {azimut - élévation} en degrés caractérisant la ligne d'horizon locale. Rappel, le SUD a pour azimut 0°.
Le format de données HADEC est un triplet {angle de déclinaison (-90° à +90°) - angle horaire du couchant (Ouest) - angle horaire du levant (Est)}.
N'indiquer que les points caractéristiques, l'interpolation fera le reste. Après vérification de la liste, l'appui sur OK provoque l'affichage du profil sur les 360° de l'horizon tel que défini.
Le titre rappelle le nom de l'horizon et les coordonnées GPS prises en compte.
Vérifier qu'il n'y a pas d'oubli ou d'erreur. A ce stade l'horizon comporte un nom et des coordonnées. Appuyer sur Fermer pour revenir au modèle de pointage.
Pas d'inquiétude : Aud'ACE a mémorisé cet horizon.
A tout moment il est possible de modifier une coordonnée ou de redessiner le graphique.

2. Elaboration d'un modèle de pointage

La méthode consiste à calculer des coefficients de correction en fonction d'écarts entre des positions réelles et des positions théoriques, il est préférable de connecter et d'initialiser le télescope ainsi que la caméra avant d'activer le menu Edition/Modifier le modèle de pointage. Une série de panneaux guide le travail.

Explication succincte du mode opératoire.
Paramétrage. L'appui sur le bouton Suivant provoque l'affichage de deux fenêtres : L'une représente l'horizon (cf. ci-dessous) et la seconde le programme de mesures (cf. suivante).
Affichage de l'horizon sélectionné avec la position théorique des zones de mesures. La zone sélectionnée dans le tableau suivant est cerclée de bleu sur le graphique.
Séquence de mesures à faire. Dans l'exemple ci-contre, une liste de 6 couples {azimut - hauteur} est proposée. Il est temps de commencer les mesures, c'est à dire une séquence comprenant :
  • Sélection d'une étoile proche de la zone visée.
  • Pointage avec le télescope.
  • Centrage de l'étoile dans le champ de la caméra.
  • Mesure des écarts.
L'appui sur le bouton #HIP d'une sélection ouvre le panneau ci-contre. Sélectionner l'une des étoiles proposées. Puis activer Pointer.
La raquette virtuelle permet le centrage de l'étoile dans le champ de la caméra. Puis activer Etoile centrée, ce qui provoque le retour au panneau de sélection de la zone à pointer.
Note : dernière chance de connecter le télescope si ce n'est pas déjà fait sinon les commandes seront désactivées.
Le bouton HIP porte le numéro de l'étoile pointée. Les décalages mesurés sont affichés dans la partie droite du panneau. Sélectionner la zone-cible suivante et recommencer le cycle.
Le graphique de l'horizon est complété avec la position observée de l'étoile (point vert cerclé de bleu).
Lorsque la séquence de mesures est complète, un bouton Calcul apparaît.
Le nom, la nature et la valeur (en arcmin) des coefficients calculés sont affichés dans la fenêtre ci-contre :
  • IH : Décalage du codeur H.
  • ID : Décalage du codeur D.
  • NP : Non perpendicularité H/D.
  • CH : Erreur de collimation.
  • ME : Désalignement N/S de l'axe polaire.
  • MA : Désalignement E/O de l'axe polaire.
  • FO : Flexion de la fourche.
  • MT : Flexion de la monture.
  • DAF : Flexion de l'axe delta.
  • TF : Flexion du tube optique.
Retour à la fenêtre principale, les résultats sont affichés. Pour appliquer le modèle à la monture, utiliser la commande Appliquer ce modèle à la monture.
Lors de la fermeture de cette fenêtre, indiquer s'il faut enregistrer le modèle de pointage pour un usage ultérieur en lui donnant un nom.
Il sera sélectionnable à l'aide du bouton Configurer de l'option Modèle de pointage dans le (bas du) panneau de configuration des télescopes.
