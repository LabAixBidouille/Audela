      PC1 : rpm -qa > maliste

      == list packages
         > rpm -qa

      == list packages contents
         > rpm -ql

      == configure network interfaces
         > man ifcfg
         > /etc/sysconfig/network/ifcfg-eth0

      == concise list of installable packages
         > zypper search 

      == get info on package   
         > zypper info 
         > rpm -qi

      == install or update package
         > zypper install

      == Mettre a jour tous les paquetages installÃ©s
         > zypper update

      == refresh package cache
         > zypper refresh

      == install build dependencies of a package   
         > zypper build-deps-install

      == find package for uninstalled file
         > webpin

      == online search packages by file
        -> packages.opensuse-community.org

      == search installable packages by name
         > zypper search

      == search installable package descriptions
         > zypper search -d

        -> sed -e 's/-[^-]\+-[^-]\+$//'
