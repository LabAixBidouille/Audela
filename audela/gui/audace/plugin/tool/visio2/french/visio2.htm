Visionneuse bis

Présentation de "Visionneuse bis"

Visionneuse bis permet de visualiser les images et les films de différents types.

Le répertoire par défaut est le répertoire sélectionné dans la configuration de Audace. Il est possible de se déplacer dans un sous-répertoire en double-cliquant sur le nom d'un sous-répertoire qui apparaît dans la liste, ou de se déplacer dans le répertoire parent en double-cliquant sur la première ligne ".."

Commandes sur la liste de fichiers

Commandes clavier

  • Flèche vers le bas : Sélectionne le fichier suivant et affiche l'image dans la fenêtre principale.
  • Flèche vers le haut : Sélectionne le fichier précédent et affiche l'image dans la fenêtre principale.
  • Escape : Efface la sélection.

Commandes avec la souris

  • Molette vers l'avant : Fait défiler la liste des fichiers vers l'avant.
  • Molette vers l'arrière : Fait défiler la liste des fichiers vers l'arrière.

Sur un nom de fichier :

  • Bouton gauche : Sélectionne un fichier et affiche l'image dans la fenêtre principale.
  • Bouton droit : Affiche le popup-menu.
  • CTRL+Bouton gauche : Ajoute/supprime une nouvelle image dans la sélection.
  • SHIFT+ Bouton gauche : Ajoute les images à la sélection.

Sur un nom de répertoire :

  • Doucle-clic sur le bouton gauche : Affiche le contenu du répertoire.

Sur un titre de colonne :

  • Bouton gauche : Trie les lignes selon l'ordre alphabétique du contenu de la colonne.
  • Bouton gauche : Modifie la largeur des colonnes.
  • Bouton droit : Ajuste la largeur de la colonne en fonction du contenu.

Popup menu de la liste des fichiers

  • Actualiser : Rafraichit la liste des fichiers.
  • Tout sélectionner : Sélectionne tous les fichiers de la liste (les répertoires ne sont pas sélectionnés).
  • Enregistrer sous : Enregistre l'image avec un format ou un nom différent.
  • Supprimer : Supprime les fichiers sélectionnés.
  • Type, Série, Date, Taille : Cocher les noms des colonnes à afficher. Par défaut, la colonne Type est la seule colonne affichée en plus de la colonne Nom.
  • Zoom : Changement du zoom de l'image.
  • Plein écran : Affichage de l'image sur tout l'écran.
  • Configurer : Permet de sélectionner les types d'images à afficher. Par défaut , seules les images FITS et GIF sont affichées.
  • Aide : Afficher cette aide.
  • Voir le film : Activer l'animation d'un fichier GIF animé ou d'un film.

Autres commandes

  • Diaporama : Affiche les images sélectionnées l'une après l'autre.
  • Connexion FTP : Affiche la liste des fichiers présents sur une autre machine connectée au réseau. L'autre machine doit être équipée d'un serveur FTP.

Configuration de "Visionneuse bis"

La fenêtre de configuration de la "Visionneuse bis" est accessible à partir du popup-menu de la liste de fichiers.

Fichiers reconnus : Sélectionner les types de fichiers qui doivent être affichés sous forme d'image ou de film

Afficher les autres fichiers : Affiche les noms des fichiers non reconnus dans la liste.

Mode Plein écran

Le mode "Plein écran" affiche les images et les films sur la totalité de l'écran.

Il peut être lancé depuis le menu "Afficher" ou depuis la "Visionneuse bis"

Commandes clavier

  • Flèche vers le bas : Affiche l'image suivante.
  • Flèche vers le haut : Affiche l'image précédente.
  • Escape : Arrête l'affichage plein écran.

Commandes avec la souris

  • Molette vers l'avant : Affiche l'image suivante
  • Molette vers l'arrière : Affiche l'image précédente
  • Bouton droit : Affiche le popup menu

Popup-menu du mode Plein écran

  • Image suivante : Affiche l'image suivante.
  • Image précédente : Affiche l'image précédente.
  • Diaporama : Affiche automatiquement l'image suivante après une durée définie dans la fenêtre de configuration du mode Plein écran.
  • Zoom : Change l'échelle d'affichage de l'image.
  • Animation : Lance le déroulement d'un image animée (par exemple image GIF animée) ou d'un film.
  • Configurer : Ouvre la fenêtre de configuration du mode Plein écran.
  • Aide : Afficher l'aide.
  • Fermer : Arrête l'affichage Plein écran.

Configuration du mode Plein écran

La fenêtre de configuration du mode Plein écran est accessible à partir du popup-menu de la fenêtre Plein écran.

Démarrer les animations automatiquement : Quand cette option est activée, les images animées ou les films sont lancés automatiquement dès leur affichage.

Couleur du fond : Cliquer sur le rectangle coloré pour changer la couleur de fond.
