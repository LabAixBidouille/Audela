Cartes du Ciel Version 3

Aud'ACE peut communiquer avec "Cartes du Ciel" pour :

- Visualiser une carte correspondant à une image affichée dans Aud'ACE.

- Pointer un objet désigné dans "Cartes du Ciel" et déplacer le télescope vers cet objet.

Au préalable, il faut configurer la communication entre Aud'ACE et Cartes du Ciel.


Configuration de "Cartes du Ciel"

  1. Ouvrir le menu Configuration --> Système.
  2. Sélectionner l'onglet Serveur.
  3. Cocher Utiliser le serveur.
  4. Cocher Garde la connexion client.
  5. Renseigner Interface réseau : 127.0.0.1
  6. Renseigner Port : 3292
  7. Enregistrer les paramètres en cliquant sur le bouton OK. C'est fini pour Cartes du Ciel.

Configuration de "Aud'ACE"

  1. Ouvrir le menu Configuration->Carte.
  2. Sélectionner l'onglet Carte du Ciel V3.
  3. Renseigner Recherche (Cartes du ciel V3) avec le chemin complet du programme Cartes du Ciel pour permettre à Aud'ACE de le lancer, s'il n'est pas déjà lancé. Vous pouvez utiliser le bouton ... (parcourir) pour cette action.
  4. Renseigner Serveur TCP/IP : 127.0.0.1
  5. Renseigner Port : 3292
  6. Enregistrer les paramètres en cliquant sur le bouton OK. C'est fini pour Aud'ACE.

Paramètres optionnels :

  • Cocher Champ fixe pour imposer un champ de taille fixe dans Cartes du Ciel. Préciser la Taille du champ(xxdxxmxxs) en renseignant la valeur en degrés, minutes et secondes d'angle. Si cette option n'est pas cochée, la taille du champ n'est pas modifiée.
    Exemple : 05d00m00s permet d'obtenir une carte avec un champ de 5 degrés.

Visualiser une carte correspondant à une image affichée dans Aud'ACE

  1. Ouvrir une image dans Aud'ACE avec le menu Fichier->Ouvrir.
  2. Sélectionner le menu Analyse->Carte, la carte correspondant à l'image est affichée dans Cartes du Ciel".

Pointer un objet désigné dans "Cartes du Ciel"

  1. Sélectionner l'onglet de votre monture dans Aud'ACE avec le menu Configuration --> Monture, cocher l'option Raquette à l'écran en choisissant la raquette superpad. Ensuite cliquer sur le bouton OK. C'est fini pour Aud'ACE.
  2. Sélectionner avec la souris l'étoile à pointer avec le télescope dans Cartes du Ciel. C'est fini pour Cartes du Ciel, il n'y a rien d'autre à faire.

  3. Dans Aud'ACE, cliquer sur le bouton Carte de la raquette SuperPad pour récupérer dans Aud'ACE les coordonnées de l'étoile sélectionnée dans Cartes du Ciel.

  4.  Le nom de l'étoile (Alp Vir pour Alpha de la Vierge) et ses coordonnées (Ascension Droite et Déclinaison) apparaissent dans la raquette.

  1. Enfin cliquer sur le bouton GOTO de la raquette pour déplacer le télescope vers l'étoile.
