Configuration de la caméra

Pour régler les paramètres de connexion d'une caméra, il faut ouvrir la fenêtre de configuration des caméras. Pour cela, vous devez aller dans le menu déroulant Configuration, puis cliquer sur Caméra... :

La fenêtre de configuration des caméras se présente sous la forme suivante :

Vous pouvez constater que le logiciel supporte un certain nombre de caméras. Il vous suffit de cliquer sur les différents onglets disponibles (Audine, Hi-SIS, SBIG, CB245, Starlight, Kitty, WebCam, AudiNet, EthernAude, TH7852A, SCR1300XTC, APN) pour explorer les possibilités de pilotage.

Dans la partie inférieure de la fenêtre ci-dessus, vous constatez la présence, derrière une case à cocher, du message suivant : Connexion automatique de cette caméra au prochain démarrage

Si après avoir coché cette case, vous appuyez sur le bouton OK ou Appliquer, à la fermeture du logiciel, celui-ci vous demandera d'enregistrer la nouvelle configuration.

Au prochain démarrage du logiciel, vous n'aurez pas à reconfigurer votre caméra. Avec cette option, la caméra Audine (dans notre exemple) se connecte automatiquement en utilisant le paramétrage précédemment choisi.

Vous pouvez vérifier ceci en notant après l'ouverture du logiciel la présence de audine (pour notre exemple) en face de Caméra dans la partie inférieure droite de la fenêtre principale (voir l'exemple ci-dessous).

Cette option est intéressante pour la plupart des utilisateurs car ils ne possèdent généralement qu'une seule caméra. Elle leur économise quelques manipulations dans les menus.

Enfin cette fenêtre comporte 4 boutons dans sa partie inférieure dont les fonctions sont les suivantes :

  • OK : Applique le paramétrage choisi pour la caméra définie par l'onglet sélectionné et ferme la fenêtre de configuration des caméras.
  • Appliquer : Applique le paramétrage choisi pour la caméra définie par l'onglet sélectionné et laisse la fenêtre de configuration des caméras ouverte.
  • Aide : Permet d'accéder directement à l'aide pour la configuration de la caméra choisie par l'onglet sélectionné.
  • Fermer : Ferme la fenêtre de configuration des caméras sans prendre en compte les paramètres modifiés.

Vous pouvez accéder au paramétrage de chaque caméra en cliquant ici.
