Zoom

Les possibilités de réglage du zoom sont les suivantes :

1. Zoom x 0,125

2. Zoom x 0,25

3. Zoom x 0,5

4. Zoom x 1

5. Zoom x 2

6. Zoom x 4

Les options x2 et x4 sont utiles pour les petites images. Appliquées à de grandes images ces options conduisent à une erreur signalée dans la Console :

# WARNING: NO MEMORY FOR DISPLAY, visu=1 set zoom=1

qui limite le zoom à la plus grande valeur possible.
