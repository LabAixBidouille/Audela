Gestion d'images FITS polychromes

Liste des fonctions utilisables :

seriesApoly liste_noms_series_monochromes nom_serie_polychrome ?options?

Création d'une série d'images polychrome à partir de plusieurs séries monochromes.

Options :

  • -rep : spécifie le répertoire dans lequel travailler.
  • -in_rep : spécifie le répertoire de la série initiale.
  • -ex_rep : spécifie le répertoire de la série finale.
  • -ext : extension des séries d'images.

polyAserie nom_serie_polychrome No_plan_couleur serie_monochrome ?options?

Extraction d'une série monochrome à partir d'une série d'images polychrome.

Options :

  • -rep : spécifie le répertoire dans lequel travailler.
  • -in_rep : spécifie le répertoire de la série initiale.
  • -ex_rep : spécifie le répertoire de la série finale.
  • -ext : extension des séries d'images.
  • -ex_polyNo : spécifie le plan couleurs de travail pour la série finale.

polyAseries nom_serie_polychrome liste_noms_series_monochromes ?options?

Conversion d'une série d'images polychrome en plusieurs séries monochromes.

Options :

  • -rep : spécifie le répertoire dans lequel travailler.
  • -in_rep : spécifie le répertoire de la série initiale.
  • -ex_rep : spécifie le répertoire de la série finale.
  • -ext : extension des séries d'images.

poly_nbcouleurs serie ?options?

Renvoie le nombre de plans couleurs de la série.

Options :

  • -rep : spécifie le répertoire dans lequel travailler.
  • -ext : extension de la série d'images.

poly_fenetre in ex coord ?options?

Fenêtrage d'une série de fichiers polychromes. La série résultante est "ex". "coord" est une liste des coordonnées de 2 coins extrêmes de la zone de fenêtrage.

Options :

  • -rep : spécifie le répertoire dans lequel travailler.
  • -in_rep : spécifie le répertoire de la série initiale.
  • -ex_rep : spécifie le répertoire de la série finale.
  • -ext : extension de la série d'images.

poly_souris_fenetre in ex ?options?

Idem serie_fenetre sauf que les coordonnées de la fenêtre sont à faire préalablement à la souris sur l'image Aud'ACE.

Options :

  • -rep : spécifie le répertoire dans lequel travailler.
  • -in_rep : spécifie le répertoire de la série initiale.
  • -ex_rep : spécifie le répertoire de la série finale.
  • -ext : extension de la série d'images.

poly_rot in ex x0 y0 angle ?options?

Transformation d'une série d'images polychromes par rotation d'angle "angle" et de centre ("x0", "y0").

Options :

  • -rep : spécifie le répertoire dans lequel travailler.
  • -in_rep : spécifie le répertoire de la série initiale.
  • -ex_rep : spécifie le répertoire de la série finale.
  • -ext : extension des fichiers..

poly_trans in ex dx dy ?options?

Transformation d'une série d'images polychromes par translation.

Options :

  • -rep : spécifie le répertoire dans lequel travailler.
  • -in_rep : spécifie le répertoire de la série initiale.
  • -ex_rep : spécifie le répertoire de la série finale.
  • -ext : extension des fichiers..

poly_series_traligne liste_des_series_initiales No_du_plancouleur_de_reference serie_finale ?options?

Registration manuelle entre plusieurs séries d'images polychromes par translation et rotation.

Options :

  • -rep : spécifie le répertoire dans lequel travailler.
  • -in_rep : spécifie le répertoire de la série initiale.
  • -ex_rep : spécifie le répertoire de la série finale.
  • -ext : extension des fichiers.
