Conversion de fichiers images au format FITS

Cette page décrit comment convertir au format FITS divers autres formats d'image. Pour cela, Aud'ACE utilise le programme "BifsConv" de Raoul Behrend. La liste des formats d'image pris en charge (notamment bmp, pic, tif, raw, ...) figure sur cette page. Ce programme est fréquemment mis à jour par Raoul Behrend, le fichier bifsconw.exe (pour Windows) ou bifsconx.exe (pour Linux) peut être téléchargé à l'adresse : ftp://obsftp.unige.ch/pub/behrend/, fichier dont la version à mettre à jour se trouve dans le sous-répertoire programme bin du répertoire d'installation d'Aude'ACE.

Liste des fonctions utilisables :

bifsconv fichier options

Lance bifsconv pour un fichier placé dans le répertoire images (le nom de fichier doit comporter l'extension). Il est possible (voire nécessaire) de préciser certaines options selon le format source (consulter à ce propos la page web de bifsconv).

bifsconv_full chemin_fichier options

Idem que bifsconv sauf qu'il faut préciser le nom complet du fichier (permet de convertir un fichier qui ne serait pas dans le répertoire images Aud'ACE par défaut).

convert_fits nom_generique extension ?répertoire?

Permet de convertir au format FITS une série d'images dont le format figure parmi ceux pris en charge par bifsconv.

convert_fits_all extension ?répertoire?

Permet de convertir au format FITS toutes les images du répertoire courant (parmi les formats pris en charge par bifsconv).

convert_fits_subdir extension ?répertoire?

Idem "convert_fits_all" avec de plus recherche de fichiers dans les sous-répertoires.

loadima_nofits fichier ?repertoire?

Chargement d'une image qui ne serait pas au format FITS (parmi ceux pris en charge par bifsconv). Le nom de fichier doit comporter l'extension. Par défaut l'image est supposée située dans le répertoire images par défaut.
