En cliquant avec le bouton gauche de la souris sur Apparence... vous avez la boîte de dialogue suivante qui s'ouvre sur votre écran :

Dans le cas présent on a accès aux couleurs diurnes de l'interface Aud'ACE, c'est à dire à des couleurs pour une utilisation de jour.

Si vous souhaitez utiliser Aud'ACE avec ses couleurs nocturnes, pour par exemple acquérir des images la nuit et ainsi ne pas être ébloui par votre écran, il vous suffit de cliquer avec le bouton gauche de votre souris sur Nocturne. Vous changez dynamiquement toutes les couleurs de l'interface Aud'ACE et vous obtenez ceci pour la boîte Apparence :

Dans tous les cas, vous devez valider votre choix par un appui sur le bouton OK ou Appliquer. Comme d'habitude, à la fermeture du logiciel il vous sera proposé de sauvegarder cette nouvelle configuration.

Bien entendu, il vous est tout à fait possible de personnaliser les couleurs de votre interface que ce soit pour une utilisation diurne ou une utilisation nocturne. Rien de plus simple. Après avoir sélectionné le mode diurne ou nocturne, il vous faut cliquer avec le bouton gauche de votre souris sur le bouton qui définit la couleur que vous souhaitez modifier. Si par exemple, vous souhaitez changer la couleur du fond général de l'interface (couleur dénomée 'backColor'), il vous suffit d'appuyer sur son bouton. Vous obtenez :

Après avoir sélectionné la nouvelle couleur, appuyez sur le bouton OK de la boîte Sélection de la couleur : backColor. La totalité de l'interface Aud'ACE se pare de cette nouvelle couleur :

Vous devez valider votre choix en appuyant sur le bouton OK ou Appliquer de la boîte Apparence.

Si ce choix ne vous convient pas, il est tout à fait possible de revenir en arrière. Vous devez quitter le logiciel Aud'ACE et répondre non à l'invitation qui vous est faite de sauvegarder cette nouvelle configuration. Au prochain démarrage, Aud'ACE se parera de ces anciennes couleurs.

Si par mégarde vous avez validé la nouvelle configuration, il vous faudra ouvrir la boîte Apparence et cliquer sur le bouton Couleurs d'origine pour revenir aux réglages 'usine'. Une nouvelle boîte de dialogue s'affiche et vous invite à rétablir ou non les couleurs d'origine.

Pour votre information voici à titre d'exemple les principales fonctions des noms des différents boutons :

  • activeBackColor : Couleur spécifique utilisée dans les tableaux bicolores.
  • activeTextColor : Couleur prise par le libellé d'un bouton quand on appuye dessus.
  • backColor : Couleur générale du fond de l'interface.
  • backColor2 : Couleur spécifique de certains fonds de l'interface.
  • canvas : Couleur des surfaces dans lesquelles s'affichent les images.
  • drag_rectangle : Couleur du rectangle tracé avec la souris pour sélectionner une zone.
  • entryBackColor : Couleur du fond d'une zone de saisie de données en mode normal.
  • entryBackColor2 : Couleur du fond d'une zone de saisie de données en mode inactif.
  • entryTextColor : Couleur du texte dans une zone de saisie de données.
  • listBox : Couleur du fond des listes déroulantes avec ascenseur.
  • textColor : Couleur générale des textes, hors zones de saisie des données.

Le plus simple est encore de changer les couleurs une à une et d'observer les évolutions dans l'interface Aud'ACE.
