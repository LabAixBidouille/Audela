1. Rotation (symbole 1/4 de tour)

Cette commande permet de faire pivoter une ou plusieurs images de 90° dans le sens horaire.

2. Rotation (symbole 1/2 de tour)

Cette commande permet de faire pivoter une ou plusieurs images de 180°.

3. Rotation (symbole -1/4 de tour)

Cette commande permet de faire pivoter une ou plusieurs images de 90° dans le sens antihoraire.
