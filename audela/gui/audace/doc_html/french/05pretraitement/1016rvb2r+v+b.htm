Conversion RVB ---> R+V+B

Le but de cette fonction est d'obtenir à partir d'une image couleur (RVB) au format FITS (3 axes) les 3 images composantes dans les bandes de couleur rouge (R), verte (V) et bleue (B), également au format FITS.

Il n'y a pas de règle particulière à suivre pour définir le nom de l'image d'entrée.

Par contre, les noms des 3 images de sortie seront par exemple :

  • nom_générique_sortie_1.fit
  • nom_générique_sortie_2.fit
  • nom_générique_sortie_3.fit
