Date de mise à jour : 08 janvier 2011

1. Tourner 90° sens horaire

Cette commande permet de faire pivoter une ou plusieurs images de 90° dans le sens horaire.

2. Tourner 90° sens anti-horaire

Cette commande permet de faire pivoter une ou plusieurs images de 90° dans le sens anti-horaire.

3. Tourner 180°

Cette commande permet de faire pivoter une ou plusieurs images de 180°.
