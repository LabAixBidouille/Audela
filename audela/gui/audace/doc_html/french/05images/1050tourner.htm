1. Tourner 90° sens horaire

Cette commande permet de faire pivoter une ou plusieurs images de 90° dans le sens horaire.

2. Tourner 90° sens anti-horaire

Cette commande permet de faire pivoter une ou plusieurs images de 90° dans le sens anti-horaire.

3. Tourner 180°

Cette commande permet de faire pivoter une ou plusieurs images de 180°.

Exécution en ligne de commande

Ces trois fonctions sont exécutables en ligne de commande dans la Console. Le répertoire images doit être celui contenant la ou les images à traiter. Le nom de la fonction ::prtr::cmdRot doit être suivi de deux listes de paramètres.

La première liste contient :

  • Le nom générique (ou la liste des noms) des images d'entrée.
  • Le chemin d'accès aux fichiers des images de sortie (mettre un point si c'est le répertoire images).
  • Le nom générique des images de sortie.
  • L'extension du ou des fichiers.
  • Le nom de l'opération à réaliser {ROT+90 | ROT-90 | ROT180}.
La seconde liste est vide. 
::prtr::cmdRot {rz_rgbr_8 . toto .fit ROT-90} {}
Elle renvoie 0 s'il n'y a pas d'erreur, sinon 1 en cas d'erreur.
