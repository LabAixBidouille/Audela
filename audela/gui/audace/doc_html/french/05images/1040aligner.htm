2. Aligner automatiquement


2.1 Principe


Cette fonction exécute automatiquement les étapes
suivantes :


    
  
  • Calcul des translations à effectuer pour que le contenu
des images se recouvrent au mieux avec une précision
inférieure au pixel.
    • 
  
  • Translation effective de chaque image.
    • 
  
  • Recadrage des images pour éliminer les zones sans
recouvrement.
    • 



2.2 Quelles images faut-il aligner ?


La réponse dépend de l'origine des images :


    
  
  • Les images issues de caméras CCD peuvent être
alignées à l'issue du prétraitement, après
correction avec les images de Précharge, de Noir et de
P.L.U.
    • 
  
  • Les images issues des APN doivent être converties en
images couleurs RVB avant d'être alignées pour
éviter les inconvénients induits par le codage sous
forme de matrice de Bayer.
    • 



2.3 Paramétrage


Il suffit de cocher les images à aligner sur l'image de
référence (à saisir dans la ligne
image_ref) et d'indiquer le nom générique des
images de sortie.




Les images recadrées portent le nom générique
suivi d'un index débutant à 1.


Après vérification de la qualité de
l'alignement, il suffit de les additionner, de faire la moyenne,
etc. avec l'une des commandes Images/Combiner.




2.4 Exécution en ligne de commande


Cette fonction est exécutable en ligne de commande dans
la Console. Le répertoire images doit être celui
contenant la ou les images à traiter. Le nom de la fonction
::prtr::cmdAligner doit être suivi de deux listes de
paramètres.


La première liste contient :


    
  
  • Le nom générique (ou la liste des noms) des images
d'entrée.
    • 
  
  • Le chemin d'accès aux fichiers des images de sortie (mettre
un point si c'est le répertoire images).
    • 
  
  • Le nom générique des images de sortie.
    • 
  
  • L'extension du ou des fichiers.
    • 


La seconde liste contient le nom de l'image de référence
(chemin complet si l'image est située dans un autre
répertoire que les images à traiter).

::prtr::cmdAligner {"trans1 trans2 trans3 trans4" . recadre .fit} {"image_ref=C:/Documents and Settings/1/Mes documents/audela/images/m45_champ.fit"}

Elle renvoie 0 s'il n'y a pas d'erreur, sinon 1 en cas d'erreur.
