Autocorrelation

The input image is processed into an output image through an autocorrelation operator.

The maximum output value of the auto-correlation process is located at the output image center.

The output of the autocorrelation processing will be displayed and saved in the current image directory as "autocorrelation.fit". If there were a file with the very same name, its content will be overridden.
The output image is encoded in a floating point format. Its dynamic range is the same as the input image, i.e. the extrema pixels values are unchanged.
